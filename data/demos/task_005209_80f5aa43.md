Set up the Rust project with `cargo new` and add `flate2` (for gzip read/write) and `encoding_rs` (for UTF-16LE decoding) to `Cargo.toml`. Per file: decompress with `flate2::read::GzDecoder`, decode the raw bytes with `encoding_rs::UTF_16LE.decode(...)` rather than assuming a BOM is present, then process line by line — for each line, find the first `;` (if any) and truncate there, trim trailing spaces/tabs (not leading — the spec only says trailing) from what remains, and skip emitting the line entirely if it's empty after that. Recompress the joined result with `flate2::write::GzEncoder` and write to `/home/user/minimized_gcode/` under the identical filename, creating that directory first. Watch the ordering of operations carefully: comment-stripping and trailing-whitespace-trimming must happen before the empty-line check, since a line that's only a comment becomes empty and must be dropped, not written as a blank line. Finally, `cd /home/user/minimized_gcode && sha256sum *.gcode.gz > /home/user/manifest.txt` so filenames in the manifest are bare, not full paths.
