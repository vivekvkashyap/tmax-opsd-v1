Treat this as a two-stage pipeline: transcription accuracy first, then automation second. For transcription, install a real ASR tool (e.g. `pip install openai-whisper` or use `whisper-cpp` with a small/base English model) rather than guessing text — run it on `/app/attacker_memo.wav`, then post-process the raw output by lowercasing and stripping all punctuation before saving to `/home/user/transcript.txt`, since the grader computes Word Error Rate against a normalized reference. Listen to or inspect the transcript for domain-like and IP-like phrases (e.g. spelled-out words like "dot", "dash") and normalize them into the exact wording your ASR produced — don't hand-correct beyond what's actually in the audio, since your file is graded verbatim. Once you have the transcript, write `/home/user/remediate.sh` as a single script that: emits the iptables DROP rule for the exfiltration IP into `firewall_rule.txt`, writes the literal CSP header string into `csp.txt`, runs `sha256sum` on the transcript into `transcript_hash.txt`, and finally `chmod 600` all four output files. Execute the script yourself so the files actually exist with correct permissions before finishing.
