Build this as a sequential pipeline of well-tested pure functions rather than one big script, since several steps feed later steps. First write the `\u` repair function: scan the raw string for every literal `\u`, check if the following 4 characters are all valid hex digits, and only if not, replace just the two-character `\u` token with `?` (leave the rest of the string untouched) — then run `json.loads` on the repaired string. Do timestamp parsing with `dateutil.parser.parse` (handles mixed formats) and format with `.isoformat()` (strip fractional seconds if present) or `strftime`. For name normalization use `.strip().title()` but be careful: Python's `.title()` lowercases letters after apostrophes/certain punctuation oddly, so test it against names containing already-uppercase runs. For country imputation, first pass: fill missing countries from the email TLD map; compute the per-country mean age (floor, using `//` or `int()` on only non-null ages) only after all countries are filled in, then do a second pass to fill missing ages using that per-country mean — order matters, don't average before imputing countries. For phone masking, use a regex like `\b\d{3}-?\d{3}-?\d{4}\b` matching exactly 10 digits with optional single hyphens, replacing with `[PHONE]`. Verify output column order and header exactly match the spec.
