Reshape the resulting 6 values into a 2x3 array in row-major order (first 3 values form row one, next 3 form row two). Finally stand up an HTTP server on `127.0.0.1:8080` with a `GET /tensor` handler that serializes this array as JSON, and leave the process running in the background so the external verifier can query it.
