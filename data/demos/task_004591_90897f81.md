Structure the Bash script as a clear pipeline of stages and test each stage on a couple of files manually first. Use `find /home/user/storage_dump -type f -print0` (null-delimited, to safely handle spaces in names) piped into a `while IFS= read -r -d '' file` loop. For MIME detection use `file --mime-type -b -- "$file"` (never trust the extension). For integrity checking, dispatch per detected type: `unzip -t -- "$file"`, `gzip -t -- "$file"`, `tar -tf -- "$file"`, checking each command's exit status to decide Valid vs Corrupt; anything not one of the three archive MIME types is `NotArchive` with SHA256 `N/A`. Compute `sha256sum -- "$file"` only for archive-type files, and before creating a hard link in `/home/user/clean_archives/`, check with `[ -e "target" ]` so you skip re-linking an existing checksum (this is the actual dedup step — hard links share inodes so no extra disk space is used). For the index, `ln -s` (not hard link) from `/home/user/archive_index/<ext>/<basename>` to the hard-linked file, preserving the original basename. Accumulate CSV rows in an array, sort by the first field with `sort -t, -k1,1`, write to a `mktemp` file, then `mv` it atomically over `/home/user/inventory_report.csv`.
