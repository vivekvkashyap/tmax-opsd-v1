Build this as a straightforward multi-stage pipeline rather than trying to do everything in one pass. First, stream `events.jsonl` line by line, `json.loads` each line, and keep only records where `event_type == "purchase"` exactly (case-sensitive, no fuzzy matching). Second, group the filtered purchases by `user_id` to get each user's purchased item set, then build an undirected `networkx.Graph()` and for every user's item set add an edge between every pair of distinct items in that set (itertools.combinations, skip self-pairs) — a user who bought the same item once only contributes to edges with their other items, not a self-loop. Third, call `nx.degree_centrality(G)` to get the normalized (divide-by-n-1) centrality for every node in one call rather than computing degree by hand. Fourth, sort items by (-centrality, item_id) to get highest-first with alphabetical tie-breaking, take the top 3, and insert them into a freshly created SQLite table using `cursor.execute("INSERT INTO item_centrality VALUES (?, ?)", (item_id, centrality))` — never string-format values into the SQL text. Create the output directory before opening the DB file.
