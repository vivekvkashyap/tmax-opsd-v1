Approach this as a forward-fill plus sliding-window computation using awk. First, filter the log for `MAX_MEM` rows only, and note the min/max Day_Index among them. Build a dense array indexed by day from min to max: iterate days in order, and whenever a day has a logged MAX_MEM value, update a "current value" variable; for days without a log entry, carry forward the last seen value (classic forward-fill/LOCF). Store these filled values in an array so you can index by day number directly. Then, starting at day (min+2), compute the 3-day rolling average as (value[N-2]+value[N-1]+value[N])/3, and the distance as the absolute difference between value[N] and that average. Use awk's printf with `%.2f` to round to exactly two decimals — be careful that naive rounding of exact halves or repeating decimals matches standard rounding behavior.
