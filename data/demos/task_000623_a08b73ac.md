Build this Rust tool in clear stages and test each before combining. First, parse `authorized_keys` line by line, splitting on whitespace to isolate the middle base64-data field (ignore key type and comment), compute SHA256 of that base64 string taken as raw UTF-8 bytes (use the `sha2` crate), hex-encode it, and compare against the set of hashes loaded from `compromised.txt` to build the compromised list. Second, port-scan `127.0.0.1:8000..=8010` by attempting a `TcpStream::connect` with a short timeout for each port — exactly one will succeed; once found, connect, send the literal payload bytes followed by a newline, and read the response until a newline or EOF. Third, base64-decode the response, convert the decoded bytes to a UTF-8 string, and SHA256-hash that decoded string's bytes (hex-encoded) — note this is a hash of the decoded (not the raw base64) text. Finally serialize a JSON object with exactly the keys `compromised_keys`, `open_port`, `decoded_response`, `response_hash` (using `serde_json`) and write it to the report path. Use `cargo add sha2 base64 serde serde_json` as needed, and verify each stage's intermediate output (compromised list, discovered port, decoded string) manually before trusting the final report.
