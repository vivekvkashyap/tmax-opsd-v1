The mathematical issue is that the periodic second-derivative (Fourier collocation) matrix has a 1-dimensional null space (constant functions), so `np.linalg.solve` fails on a singular matrix. The fix is to swap the exact solve for a least-squares solve: replace `u = np.linalg.solve(D2, f)` with `u, *_ = np.linalg.lstsq(D2, f, rcond=None)`, which returns the minimum-norm solution even when D2 is rank-deficient — do not try to regularize/perturb D2 by hand or add an artificial constraint row, since lstsq already handles the singularity correctly and is what the task asks for.
