Do the data cleaning with shell tools before touching C: use `awk -F, 'NR==1{next} {for(i=2;i<=4;i++) if($i=="") $i=0.0; print}' OFS=,` (or equivalent with `sed`) on `/home/user/data.csv` to skip the header and replace empty feature fields with `0.0`, writing a cleaned intermediate file. Write `/home/user/infer.c` to read `weights.txt` first (three comma-separated floats via `sscanf` or `strtod`+`strtok`), then read the cleaned CSV line by line parsing one int (`id`) and three floats per row (again `sscanf("%d,%f,%f,%f", ...)` or manual tokenizing), compute the dot product of the three features with the three weights, and print `id,score` with the score formatted via `printf("%d,%.1f\n", id, score)` so it always shows exactly one decimal place. Compile with `gcc -o /home/user/infer infer.c`, pipe the cleaned CSV through it, and redirect output to `/home/user/scores.csv` — remember to prepend the `id,score` header line yourself since the C program only emits data rows.
