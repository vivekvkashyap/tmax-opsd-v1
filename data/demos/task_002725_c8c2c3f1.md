Start by running `tesseract /app/query_params.png stdout` to OCR the target node ID and search depth — do not guess these values, parse them from the OCR text output. Load `nodes.csv` and `edges.csv` with the `csv` module, build an undirected `networkx.Graph` from edges (add all node_ids as nodes too, since some may be isolated), and use `nx.single_source_shortest_path_length(G, target, cutoff=depth)` to find the neighborhood — this naturally treats edges as undirected regardless of `source`/`target` column names. For the aggregate endpoint, filter edges by `interaction_type`, collect the unique `source` node ids, parse each matching node's `profile_data` JSON string with `json.loads`, average the `age` field, and round to 2 decimals with Python's `round()`. Build the HTTP server with Flask (or `http.server`) binding to `127.0.0.1:8080` exactly, and launch it as a background/daemon process (e.g. `nohup python3 server.py &` or a threaded server) so it keeps serving after your turn ends — a common mistake is running the server in the foreground and blocking, or exiting before it's confirmed listening.
