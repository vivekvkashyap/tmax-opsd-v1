Set up the venv and packages first (`python3 -m venv /home/user/bioenv && source .../bin/activate && pip install numpy biopython jupyter papermill`), then build the notebook cell by cell rather than one big script, since it will be executed headlessly via papermill and must run top-to-bottom without manual intervention. Parse the FASTA with Biopython's `SeqIO.parse`, map each base to its numeric value with a dict lookup (A=1.0, C=2.0, G=3.0, T=4.0), and compute the power spectrum as `np.abs(np.fft.fft(arr))**2`; the period-3 signal strength is the power value at the specific index `len(seq)//3` for that sequence — compute this per sequence rather than assuming which one wins, and pick the max across sequences to name `best_seq_id`. For primers, take the literal first/last 20 bases and, for the reverse primer, correctly reverse-complement the last-20-base substring (reverse the string, then complement each base via a translation table) — don't complement without reversing or vice versa. For the noise-stability check, add `np.random.normal(0, 0.5, size=len(arr))` to the original numeric array, recompute the FFT power spectrum, and compare the argmax index of power (excluding index 0 in both the original and noisy spectra) for equality. Write results to `/home/user/results.json` with the exact required keys, then run `papermill /home/user/sequence_analysis.ipynb /home/user/sequence_analysis_executed.ipynb` from inside the venv.
