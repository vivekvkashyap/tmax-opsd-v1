Scaffold with `cargo new`, add a JSON crate (e.g. `serde_json`), and implement the three checks as independent passes over the parsed edge array: Rule A rejects if any `type` isn't in the OCR'd allow-list; Rule B does a case-insensitive substring search for `DELETE`, `DROP`, `REMOVE`, `SET` inside each edge's `properties.post_hook_query`; Rule C builds a directed graph from only the `BACKUP_OF` edges and runs a cycle check (DFS with recursion-stack tracking, or Kahn's algorithm — if any node remains unprocessed, there's a cycle). Any single violation should `std::process::exit(1)`; only exit 0 if all edges pass all three rules.
