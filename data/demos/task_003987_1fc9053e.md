Build a binary format parser methodically: open each `.cfgpack` file in binary mode and use `struct.unpack` (or `int.from_bytes`) to read the spec fields in exact order — 4-byte magic `CFGP`, then a little-endian unsigned 16-bit entry count, then per-entry a little-endian u16 path length, the UTF-8 path bytes, a little-endian u32 data length, and that many raw data bytes (skip over the data, you only need the path). Advance a running offset/cursor manually so you never assume fixed entry sizes. For file discovery, use `os.walk` or `pathlib.rglob("*.cfgpack")` and filter by `os.path.getmtime` against `time.time() - 86400`. For malicious detection, check each parsed path for a leading `/` or `\`, or for `../` or `..\` substrings anywhere in the string — flag the whole file malicious if any single entry matches. Collect absolute paths into two lists, sort alphabetically, and write JSON with keys `safe_packs`/`malicious_packs`. Crucially, write to a temp file in the same directory first, then use `os.replace()` (or `shutil.move`) to atomically publish it — never write directly to the final path.
