Deduplicate by Message using a simple linear scan or hash set, keeping only the first occurrence in file order (not sorted). (3) Emit CSV with the exact header `Timestamp,ErrorCode,Message`, one row per surviving unique record, in original order. Test decode.c's bit-extraction logic against a tiny hand-crafted frame sequence before running it on the full video, since off-by-one framing errors will corrupt every downstream character.
