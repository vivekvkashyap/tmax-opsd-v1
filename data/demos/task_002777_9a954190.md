1. `python3 -m venv /home/user/venv`, activate it, `pip install numpy scipy emcee scikit-learn`.
2. Open `/home/user/vdp_mcmc.py` and replace only the body of `simulate_vdp(mu)`: build the VdP ODE right-hand side, call `scipy.integrate.solve_ivp(..., t_span=(0,10), y0=[2.0,0.0], method='BDF', args=(mu,), t_eval=[10.0])`, and return `res.y[0][-1]` if `res.success` else `None`. 3. Run the script inside the venv; confirm `/home/user/valid_mu_samples.npy` is produced and non-trivial (acceptance rate no longer near zero — sanity-check by loading it and checking it isn't empty/all-identical).
4. In `/home/user/fit_density.py`, load the samples, reshape to `(-1, 1)`, fit `KernelDensity(kernel='gaussian', bandwidth=0.5)`, call `.score_samples` on the reshaped test points `[1,2,3,4,5]` (already log-densities, no extra exp/log needed), and dump the 5 floats as a plain JSON list to the exact output path.
