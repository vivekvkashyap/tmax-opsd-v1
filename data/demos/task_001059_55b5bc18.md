Process files in a fixed order to avoid subtle bugs. First enumerate entries in artifacts_raw and separate them by type: use `find -L` cautiously — infinite symlink loops will make plain traversal hang, so instead iterate raw directory entries with `find . -maxdepth 1` (no `-L`), then for each entry that is a symlink, resolve it safely with `readlink -f` wrapped in a timeout or check `[ -e "$entry" ]` after a bounded number of `readlink` hops to detect loops/broken links, and skip those. For valid regular files and resolved symlink targets, read the first several bytes with `head -c 4 "$file" | xxd -p` (or `od`) and compare the hex prefix against each magic-number signature in order from most-specific (4-byte matches) to catch overlaps correctly, defaulting to `.bin`. Compute `sha256sum` of file contents to both determine the output filename and deduplicate — before copying, check if `<hash>.<ext>` already exists in artifacts_curated and skip if so (this also naturally dedups hard links, since identical content hashes identically). Append `<hash> <ext>` to the manifest as you go, then `sort` the manifest file at the end before finalizing.
