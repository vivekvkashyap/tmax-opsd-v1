Get Eigen3 available first (header-only, no build needed): `wget` and extract the tarball to `/home/user/eigen`, then compile against it with `-I /home/user/eigen/eigen-3.4.0`. In the C++ pipeline, process column-by-column: for imputation compute each column's mean over only non-NaN entries (check with `std::isnan`) and substitute it into the NaN cells; then, using the now-complete column, compute mean and *population* standard deviation (divide by N, not N-1) and clamp each value into `[mean-2*std, mean+2*std]` — do outlier-capping strictly after imputation, using stats recomputed on the imputed data, not the raw data. For PCA: mean-center the cleaned matrix, form the covariance matrix as `(centered^T * centered) / N` (population, matching `bias=True` conventions), and use `Eigen::SelfAdjointEigenSolver` since the covariance matrix is symmetric — note it returns eigenvalues in ascending order, so you must reverse/sort them descending and reorder the eigenvector columns to match before taking the top 2. Project the centered data onto those two eigenvectors and write the result to `/home/user/processed_data.csv` with exactly 6 decimal places. Eigenvector sign is arbitrary (±), so don't worry if your signs differ from another implementation's — magnitude is what's checked.
