First write a minimal PDB parser that reads `ATOM` lines, checks columns for record name `CA`, chain identifier `A`, and pulls the X/Y/Z fields by their fixed PDB column positions (not naive whitespace-splitting, since PDB fields can be adjacent without spaces). Compile with `-llapacke -lblas -lm`, and cross-check your program's radius against the oracle's on the same coordinate set before running it on the actual hidden PDB.
