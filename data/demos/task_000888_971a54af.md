Structure this as three clean, testable C functions: a FASTA parser (skip lines starting with '>', treat the following line as the full sequence), a sliding-window scorer that, for every valid alignment offset of the fixed-length primer against a sequence, counts matches minus mismatches and keeps the running maximum, and a stats routine computing mean and the unbiased sample variance (divide by N-1, not N) per population. Apply the scorer to every sequence in each FASTA file to build two score arrays, then compute Welch's t-statistic directly from the formula given — watch two common pitfalls: using population variance (divide by N) instead of sample variance, and forgetting that Welch's formula does NOT pool variances (don't average s_A^2 and s_B^2 together). Build with a Makefile producing `primer_test`, run it, and write the three lines to the output file using printf with `%.4f` so formatting exactly matches "Mean A: [value]" etc. Before finishing, hand-verify one or two of the per-sequence alignment scores by eye against the FASTA content to catch off-by-one window boundaries.
