Build the script in clear phases rather than one monolithic pass. (1) Walk `/home/user/backups/raw/` with `os.walk` or manual recursion using `os.path.islink()` to detect symlinks explicitly, and track visited real paths (via `os.path.realpath`) in a set so any cycle back to an already-visited real path is skipped instead of followed — never rely on `followlinks=True` without a visited-set guard, that's what causes the infinite loop. (2) For every non-symlink file, don't trust the extension — open it in binary mode and read the first few bytes to check magic numbers: `b'\x50\x4b\x03\x04'` for ZIP, `b'\x1f\x8b'` for GZIP. (3) Extract archives into a temp/staging directory (`tempfile.mkdtemp()`), then recursively re-check each extracted file's magic bytes for further archives, capping recursion depth at 3 to avoid runaway nesting. (4) Any file that is NOT an archive after this process gets copied into `/home/user/backups/clean/` with `recovered_` prepended to its basename. (5) Compute SHA256 with `hashlib.sha256(open(path,'rb').read()).hexdigest()` for each final file and write the manifest dict as JSON. Keep archives/symlinks entirely out of `clean/` and the manifest.
