In `curate.sh`, for each artifact with a `.done` marker: sort its `.partNN` files numerically (glob ordering works since they're zero-padded), concatenate their hex text together, then convert the *entire* combined hex string into raw binary — e.g. `xxd -r -p` or Python `bytes.fromhex(...)` — before compressing. This conversion step is the part a naive attempt skips: compressing the hex text itself (even with xz) stays far larger than the 850,000-byte total budget, because hex doubles the byte count and the text encoding resists binary-aware compression. Compress the *raw binary* with something strong (`xz -9`) into `/home/user/artifacts/<artifact>.bin.xz`. For metadata, convert with `iconv -f UTF-16LE -t UTF-8` first, then extract the value after `Author: ` with grep/sed/awk, and push it into Redis with `redis-cli SET meta:author:<artifact> "<value>"`. Start `redis-server` and `nginx -c /home/user/nginx.conf` as background daemons before the script needs them, and sanity-check both with `redis-cli ping` and a `curl` against `127.0.0.1:8080` for a served artifact.
