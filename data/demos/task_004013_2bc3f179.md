Work through this in order rather than jumping to the benchmark. First, find the circular import: `go build ./...` from `/app/vendored/service-mesh/server` will show which two packages (config and models) import each other; break the cycle by moving the shared type(s) into one of the packages (or a small third package) so the import graph becomes one-directional, then confirm `go build -o server .` succeeds and start it listening on port 50051. Third, write the hypothesis property test as a real gRPC round-trip against the running server, not a mock.
