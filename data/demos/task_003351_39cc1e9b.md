Build the script around a bash associative array (`declare -A seen`) that tracks timestamps already processed within the same process invocation, since state must persist only across lines of one stdin stream, not across separate script runs. Pay close attention to exact ffmpeg flag order and placement of `-ss` (before `-i` for fast seek) since output must byte-for-byte match a hidden reference implementation — small differences in flag ordering or scaling can change the produced JPEG bytes and thus the hash. Make the file executable with chmod +x and avoid buffering issues by processing line-by-line, not slurping all input first.
