Implement the standard in-place Cholesky-Crout algorithm for a symmetric 3x3 matrix: for each row i, for each column j<=i, compute L[i][j] = (A[i][j] - sum_{k<j} L[i][k]*L[j][k]) / L[j][j] for j<i, and for the diagonal L[i][i] = sqrt(A[i][i] - sum_{k<i} L[i][k]^2). Check the radicand before taking each square root (diagonal entries only) — if it is <= 1e-6, mark that matrix "Unstable" and skip further work on it, moving to the next line. For stable matrices sum all 9 entries of L as a full 3x3 (including the upper-triangle zeros you never overwrote) and print with exactly 2 decimal places using `std::fixed << std::setprecision(2)`. Parse the input file line by line, 9 floats per line, preserving order in the output. Pitfalls: forgetting to zero-initialize the upper triangle before summing; checking the sqrt argument against 0 instead of the 1e-6 threshold specified; and off-by-one errors in the Cholesky loop bounds (only compute L for j<=i, leave the rest zero).
