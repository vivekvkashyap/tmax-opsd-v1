In Python: load `/home/user/datasets/metadata.json`, and for each entry open the `source` file with `open(path, encoding=original_encoding)` to decode it correctly (a wrong encoding will silently produce mojibake that changes the SHA256 downstream), then write it back out with `open(dest, 'w', encoding='utf-8')` into `/home/user/datasets/processed/` using the original basename. Create `/home/user/datasets/aliases/` and use `os.symlink(target, link_name)` with the target being the processed file's path (absolute or correctly relative so the link actually resolves) named after each entry's `alias`. For the manifest, compute `hashlib.sha256(open(path,'rb').read()).hexdigest()` on each processed UTF-8 file, format each line as `<hash>  <basename>` (exactly two spaces), sort lines alphabetically by filename, and write to `/home/user/datasets/manifest.txt`. Double check there's no BOM or extra trailing newline being introduced during the encoding conversion that would silently shift the hash.
