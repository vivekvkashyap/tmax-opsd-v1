Write `/home/user/filter.sh` to invoke `aubiopitch` on the input wav and capture its per-frame pitch/confidence output; a genuine tonal signal should show low variance in the detected pitch and/or high confidence values across frames, while pure noise shows highly erratic pitch estimates and near-zero confidence — compute simple statistics (e.g., stddev of nonzero pitch values, or fraction of frames with confidence above some level) rather than relying on a single frame. Don't guess a threshold — empirically sweep candidate thresholds against `/app/corpus/clean/` and `/app/corpus/evil/`, checking your script's exit code against every file in both, and pick the value that yields 100% correct classification on both sets (a naive fixed threshold picked without testing is the main way this fails). Once the filter is reliable, run it over segments of `/app/sample.wav` (splitting it into chunks if needed), identify which segments are classified as valid stable tones, and log the average pitch frequency of those valid segments to `/home/user/fixture_analysis.log`.
