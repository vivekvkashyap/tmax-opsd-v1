Handle each of the three sub-tasks as a separately verifiable step. For zip-slip-safe extraction, don't just call `unzip`/`shutil.unpack_archive` blindly — iterate entries yourself (Python's `zipfile.ZipFile.infolist()` or `unzip -Z1`), for each entry compute the intended output path with `os.path.normpath(os.path.join(target_dir, entry.filename))`, and only extract it if that resolved path still starts with the target directory's absolute path (reject entries with `../` segments or absolute paths that escape it); extract safe entries individually rather than trusting the archive's own path handling. For the backup, read `settings.conf` for `TargetDirectory` (simple `grep`/`awk`/Python split on `=`), then run `tar --listed-incremental=/app/backup.snar -czvf /home/user/incremental_backup.tar.gz -C <parent-of-target> <target-basename>` and redirect tar's verbose file listing (stdout, since `-v` prints to stdout) into `backup_files.log` with `>`; note this mutates `backup.snar` in place, which is expected. For audio, use `ffmpeg -i voice_memo.wav -c:a libvorbis -ar 8000 -ac 1 -b:a 16k compressed_memo.ogg` (tune bitrate/rate downward until under the size threshold) and confirm validity with `ffprobe`.
