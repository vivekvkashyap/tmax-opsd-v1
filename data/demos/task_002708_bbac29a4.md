First extract frames from the video (e.g., with ffmpeg exporting each frame as an image, or reading frame pixel data directly via a library) and check each frame's pixels: a frame counts only if EVERY pixel is exactly RGB (255,0,0) — don't rely on average color or a sampled pixel, since a mostly-red frame with any other pixel doesn't qualify. Count how many frames pass this strict all-pixels check to get N; treat this as a value you must compute from the video, not assume. For log_filter, write a script/program that reads all of stdin as one string, counts non-overlapping occurrences of the literal substring "ERROR" (a simple substring count, not regex word-boundary matching), compares strictly greater than N, and prints exactly "ALERT" or "OK" followed by a newline — no extra whitespace or trailing output, since this will be fuzzed byte-for-byte against a hidden reference. Make the script executable (chmod +x) at the exact path. For the admin steps, create the log file and a standard logrotate stanza with daily/rotate 7/compress directives targeting the correct log path.
