Implement this as a straight, deterministic pipeline in one script rather than an interactive notebook, since exact reproducibility matters. Load X/y with `pandas.read_csv`, compute `np.linalg.cond(X.T @ X)` and write just that float to `condition.txt`. Build the `KFold(n_splits=5, shuffle=True, random_state=123)` splitter exactly as specified — the fixed seed matters because the verifier recomputes independently and compares with tolerance, so any deviation in split order will produce mismatched MSEs. For each fold, fit `LinearRegression` once and `Ridge` once per alpha in the given list, collecting per-fold validation MSEs into arrays keyed by alpha. Average each Ridge alpha's MSEs across folds and pick the alpha with the lowest mean MSE as `optimal_alpha` — don't refit only once on the full data, the CV loop is what the verifier reproduces. Run `scipy.stats.ttest_rel` between the 5 OLS fold-MSEs and the 5 optimal-Ridge fold-MSEs (paired, same fold order) to get the p-value. Write `optimal_alpha`, `ols_mean_mse`, `ridge_mean_mse`, `p_value` to `results.json` as floats. Since the verifier compares with `np.isclose`/`rtol=1e-3`, keep full float precision internally (don't round early) and only worry about matching seed/order, not manual arithmetic.
