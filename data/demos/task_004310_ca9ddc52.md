Set up a Rust binary crate in `/home/user/recommender` (use `csv`/`serde`; `ndarray` optional). Imputation: parse `train_ratings.csv`, compute each user's mean over only their originally non-empty ratings first (default 3.0 if none), then fill missing cells with that precomputed average — never impute before all averages are known, or you'll pollute later means. Build a dense matrix with users and papers each sorted numerically ascending as row/column indices, then compute pairwise cosine similarity between user row-vectors. To predict a (user,paper) rating, rank other users by similarity (ties -> smaller user_id), take the top K, and average their dense-matrix value for that paper. Sweep K in 1..=3, score each against `val_ratings.csv` via MSE, and keep the lowest-MSE K. Write `/home/user/result.txt` with `Best K: <K>`, then `Predictions:`, then rows sorted by user_id then paper_id, values rounded to 2 decimals. Build with `cargo build --release`, run it, and inspect `result.txt` yourself before finishing.
