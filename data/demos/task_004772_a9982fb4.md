For automating `/app/init_env.sh`, write an `expect` script that spawns the script, expects the literal "Username:" and "PIN:" prompts, and sends `sysadmin` and `8080` in order. For the supervisor, use a `while true; do ... ; done` loop around the binary reading from a `mkfifo`'d named pipe, redirecting stdout to the log, so it auto-restarts on exit. For logrotate, write a config block with `daily`, `rotate 7`, `compress`, `missingok`, and `size 10k` (or `minsize 10k`) targeting the log path.
