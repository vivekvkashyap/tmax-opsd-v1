Parse `networks.txt` line by line: split on whitespace into GraphID, NumNodes, and the edge list, then split the edge list on `;` and each edge on `,` to build an adjacency list for an undirected graph. Implement the SI-model simulation per graph: track infection status per node starting with only node 0 infected; each discrete time step, for every susceptible node, independently roll infection with probability 0.3 for *each* infected neighbor it currently has (i.e., don't just infect it once if any neighbor is infected — the note about "k independent rolls" matters and affects the mean time), and stop when all nodes are infected, recording that step count as T. Run this 5000 times per graph with the `rand` crate and average T. A common mistake is updating infection status in-place mid-step (causing newly-infected nodes to immediately infect others in the same step) — snapshot which nodes are infected at the start of the step and only apply new infections at the step's end. Finally write the per-graph means, ordered by GraphID ascending, into a `/mean_times` f64 dataset in `/home/user/training_data.h5` via the `hdf5` crate, and run the binary to produce the file.
