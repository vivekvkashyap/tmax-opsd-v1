Extract the tar with symlink-loop protection: use `tar --extract -f repo_dump.tar -C /home/user/extracted_repo/` and be aware that plain `tar` can create the symlinks fine (creating a loop doesn't itself hang tar), but any later recursive walk (`find`, `cp -r`, a naive du) can hang on the loop — use `find ... -maxdepth` guards or `find -type l` to enumerate and remove symlinks before doing any recursive traversal, and always identify symlinks with `find -type l -delete` (not `-follow`). For each remaining regular file, read the first bytes with `xxd`/`od -An -tx1 -N8` or Python `open(f,'rb').read(8)` and compare against the three magic-byte signatures to decide the extension; rename in place (`mv name name.ext`), keeping the original base name intact. Build the final `.tar.gz` with only `*.elf` files at archive root using `tar -czf out.tar.gz -C dir file1.elf file2.elf ...` (the `-C` plus bare filenames avoids embedding directory paths). Compute `sha256sum *.elf`, then sort lines alphabetically by hash (not filename) — `sort` on the sha256sum output sorts correctly since the hash is the first field.
