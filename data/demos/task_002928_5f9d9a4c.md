Use a fresh/temp Kuzu database directory per invocation (e.g. `tempfile.mkdtemp()`) and clean it up — reusing a stale DB directory across runs on the hidden corpus will cause schema-creation errors or leaked state that breaks the 100%-accuracy requirement.
