After extracting the tar.gz, for each entry in mapping.json read the source file's bytes and decode using the `encoding` field given in the mapping (don't assume UTF-8 in, only UTF-8 out) — e.g. in Python, open with `encoding=mapping[key]["encoding"]` or decode raw bytes with `.decode(encoding)`, then re-encode/write as UTF-8. Write outputs to `processed/` under their `target_name`, then zip them with `zipfile.ZipFile` writing each file by its basename only (`arcname=filename`) so the archive has no parent directory. Verify by unzipping and re-reading as UTF-8 to catch any residual encoding mismatch.
