Build the tokenizer as a single pipe chain: lowercase with `tr 'A-Z' 'a-z'` (or `tr '[:upper:]' '[:lower:]'`), then replace every non-alphanumeric character with a space using `tr -c 'a-z0-9' ' '` (the `-c` complements the set) or `sed 's/[^a-z0-9]/ /g'`, then split on whitespace into one token per line with `tr -s ' ' '\n'` or `tr ' ' '\n'` followed by squeezing, and finally drop empty lines with `grep -v '^[[:space:]]*$'` (or `grep .`). Chain these in the exact stated order since later steps depend on earlier ones (e.g. splitting before blank-line removal). For bootstrap.sh, read the three positional args (`$1` file, `$2` n, `$3` seed) and implement sampling with-replacement in a single gawk invocation: seed via `srand(seed)` in a BEGIN block is wrong if placed before reading NR — instead load all lines into an array during the main body, then in an END block loop `n` times picking `idx = int(rand()*NR) + 1` and printing `lines[idx]`, calling `srand(seed)` once in BEGIN. Verify determinism by running the script twice with the same seed and diffing outputs before generating the three required samples and their sha256sum manifest.
