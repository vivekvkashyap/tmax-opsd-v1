Only the first occurrence of a given (timestamp, normalized_key) pair should be emitted; increment a duplicate counter for the rest. Emit surviving records to stdout in the same 3-column format using the *normalized* key, and after all input is consumed, print the stats line to stderr in the exact literal format `Processed: <N> lines, Duplicates dropped: <M> lines`. Compile with `g++ -O2` and run it redirecting stdin/stdout/stderr to the specified files, then diff your output against manual reasoning about the input to catch off-by-one counting errors.
