Use Python's `configparser` to read `/home/user/dataset/config.ini` and get the log/bin filenames per experiment section. For each experiment, read its log file and scan for the multi-line success pattern: split into lines, and check whether any line is exactly `STATUS: SUCCESS` (strip whitespace, compare exactly) AND the very next line is exactly `QUALITY: HIGH` — a naive check that just looks for both strings present anywhere in the file (not adjacent) will wrongly pass a log where they're separated, so iterate with an index and compare `lines[i]` and `lines[i+1]`. For successful experiments only, build a tar.gz with Python's `tarfile` module (`mode="w:gz"`), adding each file via `tar.add(src_path, arcname=f"{exp_id}/{filename}")` — using `arcname` is the key step to avoid embedding the absolute `/home/user/...` path inside the archive. Run the script afterward so the archive actually exists at `/home/user/curated_archive.tar.gz`.
