For the checker script, never extract untrusted tarballs directly onto the filesystem: first inspect the archive listing (`tar -tvf`) and iterate over entry names/link targets programmatically (Python's `tarfile` module is easiest) to reject any entry whose name is absolute, contains `..` segments that escape the root, or whose symlink target is absolute — do this membership check before ever calling `.extractall()`; if you do need to inspect config content, extract only that one member into a locked-down `mktemp -d` sandbox. For the policy check, parse `app_config.json`'s `mount_path` and reject if it equals the denied path or is a subdirectory of it (path-prefix comparison on normalized path segments, not a naive string `startswith`, to avoid false positives like `/media/usb` vs `/media/usb_backup`). Exit 0 only if all checks pass, exit 1 on any single violation.
