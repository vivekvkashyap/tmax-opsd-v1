Install OpenMPI and mpi4py first (`apt-get install -y libopenmpi-dev openmpi-bin` then `pip install mpi4py`, using sudo as needed). For the integrator, split the (N+1) y-grid points into contiguous row chunks across ranks (e.g. via `numpy.array_split`), and on each rank compute the 2D composite-trapezoidal contribution for its rows using vectorized numpy (weight 0.5 at the global y=0/y=N boundary rows and at x=0/x=N columns within every row, matching the standard trapezoidal rule) — the key pitfall is double-counting or dropping the boundary weighting at rank boundaries, so decompose by *point ownership*, not by naive overlapping intervals. Sum with `comm.reduce(partial, op=MPI.SUM, root=0)` and have only rank 0 print the final float. Drive it with `mpirun -np 4 python3 parallel_integrate.py --N <value>` from `run_study.sh` for N=100,200,400,800, capturing each stdout value into the JSON keyed by N as a string.
