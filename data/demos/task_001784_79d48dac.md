For each file, open it and wrap it with `memmap2::Mmap::map(&file)` — hash the mmap'd byte slice with `sha2::Sha256`, not a manual `read()` loop, since the task explicitly requires memory-mapped I/O. Create the `docs_packaged` output directory first if missing, then `cargo build --release` and run the binary once to produce both files.
