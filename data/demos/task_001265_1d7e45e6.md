Do this as a clear sequence of passes rather than one big script. First extract the outer tar.gz into `/home/user/docs_processed/`, then recursively walk the tree with `find` looking for `.zip`/`.tar.gz`/etc., extract each into its containing directory (`unzip -d <dir>` / `tar -xzf ... -C <dir>`), and delete the nested archive file afterward — repeat the walk if extraction can produce further nested archives. Next, for every regular file: if its name ends in `.txt.old` or `.md.draft`, strip that suffix and rename to `<base>.md`; separately lowercase the basename and replace spaces with `-` (do both transforms together, e.g. via `mv "$f" "$(dirname "$f")/$(basename "$f" | tr 'A-Z ' 'a-z-')"` after fixing the extension) — do extension-fixing before/independently of case/space fixing so both rules apply cleanly to every file. Then run `sed -i` across all `.md` files to replace `AcmeCorp`->`GlobalTech` and `Draft: Yes`->`Status: Published`. Build the manifest by `find`-ing all `.md` files, computing `sha256sum` for each, stripping the `docs_processed/` prefix, and sorting lines by path before writing. Finally `tar -czf .../published_docs.tar.gz -C /home/user/docs_processed .` (note the `-C` plus `.` so no wrapper directory is embedded).
