Unpack in two passes: `tar -xf telemetry_logs.tar` to get the nested `.tar.gz` files, then `tar -xzf` each of those to recover the raw `.log` files. Filter each log with `grep -vE '\[TRACE\]|\[DEBUG\]'` (exact bracketed substrings, other lines untouched) and write results into `/home/user/cleaned_logs/`, preserving original filenames. Count lines of each cleaned file with `wc -l` and append `filename,count` to the inventory — but every append must be wrapped in an exclusive lock since a background worker writes concurrently: e.g. `( flock -x 200; echo "$line" >> /home/user/inventory.txt ) 200>>/home/user/inventory.txt.lock`, or use `flock -x 9 9>>/home/user/inventory.txt.lock` around the whole write block. Don't just append normally and hope for no interleaving — the grader specifically checks your scripts/history for `flock` (or `fcntl` if using Python) applied around the inventory writes. Verify each cleaned file's line count with `wc -l` yourself before trusting the numbers you record.
