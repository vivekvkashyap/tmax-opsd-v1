Start by extracting the two port numbers from `/app/diagram.png` with OCR (e.g. `tesseract /app/diagram.png -`), then read the printed "Gateway Port" and "Internal Storage Port" values from the OCR text rather than guessing — verify the OCR output looks sane before trusting it. Set up the forward with `socat TCP-LISTEN:<gateway-port>,fork,reuseaddr TCP:127.0.0.1:<internal-port>` (or an equivalent Python relay) and confirm it's actually listening (`ss -ltnp` or a quick `nc` test) before running your download script. In `process_backup.py`, connect a raw TCP socket to localhost on the gateway port and loop `recv()` until the connection closes (EOF), buffering to a temp file rather than assuming a fixed size. The key to the compression metric is the qemu-img invocation: convert the raw temp file to qcow2 with `qemu-img convert -f raw -O qcow2 -c <src> /home/user/archived_disk.qcow2` — the `-c` flag enables compression, and because the source is sparse (mostly zeros), qcow2's cluster allocation will naturally skip the zero regions, so verify the resulting file size and `qemu-img info` output before declaring done.
