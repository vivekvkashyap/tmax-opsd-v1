Write a Go walker that manually recurses rather than using `filepath.Walk` (which doesn't follow symlinks) or naive recursion (which loops forever). At each directory entry, `os.Lstat` first to detect symlinks; when a symlink points to a directory, resolve it with `filepath.EvalSymlinks`, then check whether you've already visited that resolved path (e.g. keep a `map[string]bool` of resolved directory paths, or track device+inode pairs from `syscall.Stat_t`) — if seen, skip it to break the loop, otherwise descend into it. For files (including symlinks to files), open and read the first 4 bytes and compare exactly against the magic byte sequence given in the spec before treating it as a valid backup chunk; skip files shorter than 4 bytes. For each valid file, resolve its absolute canonical path and compute a SHA-256 over its full contents. Collect all matches, sort the slice by `path` string before marshaling, and write indented JSON to the manifest path with the exact `files`/`path`/`checksum` field names. Compile with `go build` and run it, then diff the output structure against the spec.
