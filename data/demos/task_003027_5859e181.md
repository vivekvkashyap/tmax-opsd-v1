To defeat path traversal, strip directory components with `basename` on the extracted path (never trust the raw path, since entries deliberately contain things like `../../../home/user/.ssh/authorized_keys`) so every output lands directly under `/home/user/extracted_logs/`. Convert each payload's bytes from its declared source encoding to UTF-8 with `iconv -f <encoding> -t UTF-8`. Write the converted content to a `<basename>.tmp` file first, then `mv` it to the final basename only after the write completes, so a concurrent reader never sees a partial file. After each successful move, append `EXTRACTED: <basename>` to `/home/user/extraction_audit.log`, wrapping the append in `flock` (e.g. `flock /home/user/extraction_audit.log -c 'echo ... >> ...'` or a file-descriptor-based flock) so concurrent writers don't interleave. Create the output directory with `mkdir -p` first, then run the script and inspect the resulting files and log for correctness.
