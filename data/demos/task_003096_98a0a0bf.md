Because standard `zip`/`Info-ZIP` CLI tools normalize/strip `../` from paths, you cannot use the plain `zip` command to embed traversal paths — instead build the archive programmatically with Python's `zipfile` module (or by hand-writing ZIP local/central-directory headers) so you can write entries whose names literally contain `../system_root/...` unmodified.
