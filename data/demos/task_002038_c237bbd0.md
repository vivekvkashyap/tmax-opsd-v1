Build a small pipeline that applies IIR notch filters (`scipy.signal.iirnotch`) at each detected peak frequency plus a mild bandpass/lowpass or spectral-gate step for the hiss, then measure MSE against `val_clean.wav` after normalizing both to the same scale. Loop over several parameter combinations (notch Q factor, filter order/cutoffs, at least 5 trials) logging each trial's parameters and resulting MSE to `/app/experiments.csv`, and keep the lowest-MSE config. Encode that winning config directly into `/app/run_pipeline.sh` (positional args: input wav, output wav) using the same filter chain, run it on `test_noisy.wav`, and confirm the output is written back as 16-bit PCM at the original sample rate.
