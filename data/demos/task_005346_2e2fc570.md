Build this as a real running HTTP service, not a script that computes things once — the verifier issues live requests. Start by confirming connectivity to Postgres (`psql -h 127.0.0.1 -U admin -d telemetry`) and Redis (`redis-cli -h 127.0.0.1 ping`) before writing app code. In `/engineer_features`: load `raw_signals` into a dataframe, compute the pairwise Pearson correlation matrix, and for any pair exceeding 0.90 drop the higher-indexed column (process pairs in a stable index order so drops are deterministic and don't cascade incorrectly); impute remaining NaNs with per-column mean; fit Ridge(alpha=1.0) inside 3-fold CV and rank features by mean absolute coefficient across folds to pick the top 5. Persist the 5 names to the JSON file and cache the reduced dataset (id, target, 5 features) in Redis as a JSON string under the exact key. Enforce the `X-API-Token` header check as middleware/decorator on every route, returning 401/403 when absent or wrong, and test it yourself with `curl` (with and without the header) before declaring done. Leave the process running in the background bound to 127.0.0.1:8080.
