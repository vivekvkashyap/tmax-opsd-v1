Parse the single-line CSV by splitting on commas and reshaping the flat list into (x,y) pairs (even indices are x, odd are y). Build the 4-column Vandermonde matrix (columns 1, x, x^2, x^3) for each data point, then solve the normal-equations / least-squares system with `gsl_multifit_linear` (or equivalent QR-based solver) rather than manually inverting X^T X, which is numerically fragile for small samples. Extract the resulting coefficient vector as beta_0..beta_3 and write them with `fprintf` using exactly `"%.4f,%.4f,%.4f,%.4f\n"` — the verifier does an exact string comparison after trimming whitespace, so don't add extra formatting, labels, or scientific notation. Compile with `g++` isn't right here — use `gcc`/`g++` per the toolchain but link GSL properly (`-lgsl -lgslcblas -lm`), then actually execute the binary so `/home/user/coefficients.txt` is written to disk. Sanity-check your fit by verifying the residuals are near zero on the given points before considering the task done.
