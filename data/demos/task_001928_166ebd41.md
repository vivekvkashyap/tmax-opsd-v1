Make pipeline.sh executable with chmod +x. Finally add the GitHub Actions workflow with `on: push` to main, one job `test-pipeline` on ubuntu-latest, a Python 3.9 setup step, a step running `bash pipeline.sh`, and a final step that fails the job if diff_report.txt is non-empty (e.g. `[ -s diff_report.txt ] && exit 1 || exit 0`).
