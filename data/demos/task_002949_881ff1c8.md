Convert the manifest with `iconv -f UTF-16LE -t UTF-8 incoming_manifest.log -o manifest_utf8.log`, watching for a leading BOM that iconv/your parser may need to strip. Parse the multi-line `[Artifact]` blocks with a small state-machine (awk, or Python reading line by line and resetting on the `[Artifact]` marker) that pairs each `Temp-Name:` with the following `Real-Name:` value — don't assume a single-line regex will work since records span multiple lines. Iterate the extracted mappings and `mv`/rename each matching file in `/home/user/artifacts`. When building the tar, `cd` into the artifacts directory first (or use `-C`) so the archive stores bare filenames at its root rather than full paths — a plain `tar -cf out.tar path/to/*.tar.gz` will embed directory components. Finally list the tar's contents and sort alphabetically into final_list.txt. Double-check the renamed files actually exist before tarring, since a parsing bug would silently produce an incomplete or wrong archive.
