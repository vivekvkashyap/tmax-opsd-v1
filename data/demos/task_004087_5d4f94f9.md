Build this as a two-stage pipeline glued together by a bash wrapper. In the bash script: install deps (e.g. `pip install scikit-learn numpy opencv-python-headless scipy`), then run `ffmpeg -i /app/experiment_record.mp4 -vf fps=1 <tmpdir>/frame_%05d.png` (or similar padded pattern) so frames sort correctly by filename — never rely on ffmpeg's default frame numbering being unpadded, use a wide zero-padded pattern so lexicographic sort equals chronological order. In the Python script, glob and sort the frame filenames explicitly, load each grayscale (e.g. `cv2.imread(path, cv2.IMREAD_GRAYSCALE)` or PIL 'L' mode) and flatten to a 1D vector, stack into an (n_frames, n_pixels) matrix. Fit `sklearn.decomposition.PCA(n_components=5)` and transform, then fit `sklearn.mixture.BayesianGaussianMixture(n_components=3, covariance_type='full', random_state=42)` on the 5D features. Use `predict` (max posterior) for cluster assignment, not `predict_proba` argmax by hand unless you verify they match. Write frame_index (0-indexed, matching sorted frame order) and phase_cluster to CSV with the exact header. Do not hardcode frame counts anywhere — derive everything from the glob results.
