Process archives one at a time in a loop, gating everything on integrity first: `tar -tzf "$f" >/dev/null 2>&1` returns nonzero for a corrupted archive — skip it entirely (don't extract configs or logs from it) and only proceed with archives that pass this check. For flattening configs from valid archives, extract with `tar -xzf "$f" -C /tmp/extract_dir` then `find /tmp/extract_dir -name '*config.json' -exec cp {} /home/user/clean_configs/ \;` (cp strips the directory automatically since you're copying by basename into a flat target dir), or use `tar --wildcards --transform 's,.*/,,' -xzf "$f" '*config.json' -C /home/user/clean_configs/` directly. For the compressed logs, avoid writing the uncompressed `.log` to disk — pipe with `tar -xzOf "$f" '*.log.gz' | gunzip | grep CRITICAL >> /home/user/critical_errors.log` (extract-to-stdout, decompress-to-stdout, grep, append) per valid archive. Sort the collected log at the end as instructed. Finally `tar -czf /home/user/consolidated_configs.tar.gz -C /home/user/clean_configs .` so paths inside the tarball are rooted at the files themselves, not the directory.
