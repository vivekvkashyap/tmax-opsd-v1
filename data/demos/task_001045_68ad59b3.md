Sequence the work as reconstruct -> safe-extract -> dedup -> manifest. Reconstruct: list the `.partN` files and concatenate them in ascending *numeric* order of N (sort as integers, not lexicographically, so part10 doesn't sort before part2) into one binary, then open it with `tarfile`. Safe extraction: never call `tar.extractall()` blindly — iterate `tar.getmembers()`, for each member resolve the intended output path (`os.path.join(dest, member.name)`), normalize it (`os.path.realpath`), and only extract if that resolved path is still inside the destination directory (`os.path.commonpath` or `startswith` check on the normalized dest); silently skip anything that resolves outside, including absolute-path entries and `../` traversal. Dedup: after extraction, hash every file's content (sha256), group files with identical hashes, and within each group keep the alphabetically-first relative path as the canonical file, replacing the rest with `os.symlink` using a *relative* target computed via `os.path.relpath` from the symlink's own directory. Manifest: walk the tree, emit `type`/`sha256`/`target` per relative path exactly as specified, with symlinks having `sha256: null` and files having `target: null`.
