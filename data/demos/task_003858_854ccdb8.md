Work the pipeline in discrete stages and check each one before moving on. First extract every `Authorization: Bearer <token>` value with `grep -oP`. For each JWT, split on `.` and base64url-decode the header segment: JWT base64url omits padding and uses `-`/`_` instead of `+`/`/`, so translate characters and pad with `=` to a multiple of 4 before feeding to `base64 -d` (or use `python3 -c` with `base64.urlsafe_b64decode`) — a naive `base64 -d` on the raw segment will fail or silently produce garbage. Use `jq -r '.alg'` (or grep -i) case-insensitively against the decoded header JSON to flag `alg` == "none" in any casing. For flagged tokens, decode the payload segment the same way and pull `.sub` with `jq`. Dedupe with `sort -u`, write to the plaintext temp file, then encrypt with `openssl enc -aes-256-cbc -salt -pbkdf2 -pass pass:NetSec2023` (older openssl may reject `-pbkdf2`, so verify openssl's version behavior in your test). After encrypting, `chmod 600` the `.enc` file and `rm` the plaintext — order matters, don't delete the plaintext before encryption succeeds.
