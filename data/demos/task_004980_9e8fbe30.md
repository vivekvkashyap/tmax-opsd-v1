Work in order: first run `tesseract /app/architecture.png -` (or convert to a higher-res image first if OCR is noisy) to recover the Web Port, Auth Token, and Mock SSH Port — do not guess these values. Build the HTTP server (plain Python `http.server`/`socketserver` is simplest) that checks the raw `Authorization` header against `Bearer <token>` exactly, returning 200/`M1CRO_OK` on match and 401 otherwise — remember HTTP headers are case-insensitive on the name but the value must match verbatim. For the mock SSH port, a small Python or `socat`/nc-based TCP listener that writes the greeting bytes then closes the socket is easiest; test with `nc 127.0.0.1 <port>` to confirm exact bytes including `\r\n`. For the git hook: `git init --bare /home/user/micro_repo.git`, then write `hooks/pre-receive` reading stdin (`old new ref` lines), using `git log <old>..<new>` (or diffing against empty tree for new refs) to pull commit messages, and grep for the literal word `APPROVED`; exit nonzero to reject. Put both long-running services in `start_services.sh`, launched with `&` and `disown`/`nohup`, verify each is listening via `ss -ltn` before finishing, and run everything as your own user (no sudo).
