Since ISO-8859-1 maps each byte 1:1 to the Unicode code point 0-255, read the file as raw bytes (`std::fs::read`, not `read_to_string` which would fail/mis-decode on the non-UTF-8 byte), then build a Rust `String` by mapping each `u8` byte through `char::from_u32(byte as u32)` (always succeeds for 0-255) and collecting into a String — this correctly turns 0xE9 into the two-byte UTF-8 encoding of U+00E9. Do the `[SECRET_ID]` → `[REDACTED]` substitution on that String with `.replace(...)`. For atomicity, write the result to a temp file in the same directory (e.g. `/home/user/staging/clean_system.tmp`) using `std::fs::File`, call `.flush()` (and consider `.sync_all()`), then rename with `std::fs::rename(tmp, final)` — same-filesystem rename is atomic, but only if the temp file lives in the same directory as the destination, so don't use a system tmp dir. Compile directly with `rustc process_log.rs -o process_log` (no cargo) and run it once to produce the output file.
