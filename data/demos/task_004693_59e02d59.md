In `/home/user/infer.py`, POST the CLI argument as `{"text": ...}` to the tokenize endpoint and pull out `tokens`; if empty, print `0.0000` and exit immediately. Compute token lengths, mean, and sample stddev with `ddof=1` (Bessel's correction, i.e. divide sum of squared deviations by N-1, not N) — guard N==1 by setting margin-of-error to 0 instead of dividing by zero. Compute the 95% CI upper bound with the given Z=1.96 formula and short-circuit to `-1.0000` if it exceeds 12.0. Otherwise connect to Redis (`redis.Redis(host='127.0.0.1', port=6379, db=0, decode_responses=True)`), fetch `prior_A` (default -0.5) and for each token fetch `token_A:<token>` (default -5.0 if missing), sum them all, and print with `f"{value:.4f}"` — exact formatting matters since the grader does string/byte comparison against a reference run over hundreds of random inputs. Make the script executable with the correct shebang.
