Install `pandas`, `tiktoken`, and `matplotlib` first (e.g. `pip install pandas tiktoken matplotlib`). In the script, set the matplotlib backend to `Agg` via `matplotlib.use('Agg')` *before* importing `matplotlib.pyplot`, since a headless environment has no display and a GUI backend will crash or hang. Read `/home/user/data/reviews.csv` with `pandas.read_csv`, which strips the outer CSV quoting so you tokenize the actual review text (not the surrounding quote characters) — do not hand-parse the CSV yourself, since manual quote-handling is a common source of off-by-a-few-tokens errors. Use `tiktoken.get_encoding("cl100k_base")` and call `.encode(text)` per row, timing only the tokenization loop (not the file read or JSON/plot writing) with `time.perf_counter()` immediately before/after. Sum the per-row token counts (`len(encoding)`) for `total_tokens`, write `{"total_tokens": ..., "time_seconds": ...}` to `/home/user/results/metrics.json`, then plot a histogram of the per-review token counts and save it to `/home/user/results/token_dist.png` via `plt.savefig`, closing the figure afterward.
