Start by extracting frames losslessly: `ffmpeg -i /app/experiment_data.mp4 -pix_fmt gray frames/frame_%04d.pgm` (PGM is a trivial text/binary header format you can parse by hand in C++ — skip the `P5\nW H\nMAXVAL\n` header then read raw bytes). In C++, for each 64x64 frame, identify "active" pixels above some intensity threshold relative to background noise, then compute a spatial statistic that distinguishes a tight cluster from scattered noise — e.g. the covariance matrix (or just the variance/spread) of the (x,y) coordinates of active pixels weighted by intensity: a real cluster has small spatial spread and high concentrated intensity, while noise pixels are spread uniformly across the 64x64 grid. A simple, robust score is inverse-spread times mean intensity of active pixels, or the trace/determinant of the 2x2 covariance matrix of active-pixel coordinates. Compile with `g++ -O2`, run it over all 200 frames, and write `frame_id,score` rows to /home/user/anomaly_scores.csv (0-indexed, no extra columns). Sanity-check by eyeballing which frames get high scores versus low — clusters should visibly separate from the noise-only frames.
