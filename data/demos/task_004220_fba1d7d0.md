Once the binary reliably exits 0/1 on good/bad YAML, write the `pre-receive` hook: read the three whitespace-separated fields per line from stdin, use `git diff --name-only --diff-filter=AM <old> <new> -- '*.yaml'` (or equivalent) to list changed YAML files, extract each file's blob content with `git show <new>:<path>`, pipe it to the compiler and check its exit code, AND separately grep/parse the YAML yourself for any `hostPath` block whose `path:` value is `/var/run/docker.sock` or `/var/run/containerd.sock` — reject (exit nonzero, no stdout needed) if either check fails on any file, otherwise exit 0. Make the hook executable and test it manually against both corpora before trusting it.
