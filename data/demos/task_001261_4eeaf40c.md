Build this as four discrete artifacts, each independently testable. dedup.py: read the CSV, for each row compute `hashlib.md5(f"{timestamp},{sensor_id},{value}".encode()).hexdigest()`, keep only the first row seen per hash (use a seen-set, preserve first occurrence), then sort the surviving rows by (sensor_id, timestamp) before writing with the original header — sorting after dedup, not before, matters since "first occurrence" is defined on the raw feed order. compute_metrics.py: load the baseline JSON list (24 floats), group the deduped rows by sensor_id in chronological order, compute Euclidean distance (`sqrt(sum((v-b)**2))`) between each sensor's 24 values and the baseline, round to 2 decimals, and write `sensor_id,distance` sorted alphabetically by sensor_id. run_pipeline.sh: `python3 dedup.py && python3 compute_metrics.py`, then `chmod +x` it. Finally install a crontab line `0 0 * * * /home/user/run_pipeline.sh` via `(crontab -l 2>/dev/null; echo "0 0 * * * /home/user/run_pipeline.sh") | crontab -`, then confirm with `crontab -l`.
