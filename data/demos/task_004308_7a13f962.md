Then write a Go server using only `net/http` (no external deps needed) listening on `127.0.0.1:9090` with three handlers: `/montecarlo` draws `samples` values from `rand.NormFloat64()*sigma+mu` and returns the empirical fraction exceeding threshold as `{"mc_prob":...}`; `/integrate` computes the exact tail probability via the normal CDF using `math.Erfc((threshold-mu)/(sigma*math.Sqrt2))/2` and returns `{"exact_prob":...}`; `/test` reads `/app/profiling_data.csv` (one float per line, no header), computes the sample mean and N, then Z = (sampleMean-mu)/(sigma/sqrt(N)), returning `{"z_score":...}`. Compile the Go binary, launch it in the background (e.g. `nohup ./server & `), and curl each endpoint yourself to confirm the JSON shape and that the server stays up before finishing.
