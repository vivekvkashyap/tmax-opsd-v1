First handle the file operations: use `find /app/incoming -name '*.dat' -mtime -7` to locate recently modified files, then rename each to `.car`. For the image, run `tesseract /app/policy.png stdout` (installing tesseract-ocr if needed) and read off the "Base Directory" line to get the extraction root — do not hardcode a guessed path. For the sanitizer script, the core technique is: for each entry's "path", compute `os.path.abspath(os.path.join(BASE_DIR, path))`, then use `os.path.commonpath([BASE_DIR, resolved])` and check it equals `BASE_DIR` (and that resolved != BASE_DIR itself) before accepting it; otherwise print `REJECTED: <original_path>`. The naive mistakes are: (a) using a plain string `startswith` check instead of `commonpath`, which lets a sibling directory like `/app/artifact_repo/safe_zone_hacked` pass incorrectly; and (b) forgetting that `os.path.join` discards the base when the second argument is an absolute path (e.g. `/etc/passwd`), which is actually fine since the resulting absolute path will naturally fail the commonpath check. Preserve input order and handle empty/malformed entries gracefully since a fuzz harness with random `../`, `./`, and absolute-path segments will test thousands of cases against a hidden reference implementation.
