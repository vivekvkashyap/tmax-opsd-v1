Install the MPI toolchain first (`apt-get install -y openmpi-bin libopenmpi-dev`, `pip install mpi4py`) since compilation of mpi4py needs the dev headers. Structure the script around `mpi4py.MPI.COMM_WORLD`: get `rank` and `size`, then split `N` into contiguous chunks so every rank knows exactly which interval indices `[start, end)` it owns — use integer division with the remainder distributed to the first few ranks so the split is exact and non-overlapping (avoid naive `N // size` alone, which drops leftover intervals). Each rank computes the Midpoint Rule sum only over its own chunk: `f(x_i) = 4/(1+x_i^2)` at `x_i = (i + 0.5) * (1/N)`, accumulate a local sum, then multiply by `dx` only once (either locally before reducing, or reduce the raw sum and multiply by `dx` on rank 0 — pick one and be consistent). Use `comm.Reduce(local_sum, ..., op=MPI.SUM, root=0)` to combine, and have only rank 0 compute the error against `math.pi` and write the three-line output file. Run with `mpiexec --allow-run-as-root -n 4 python3 ...` and confirm the output file appears with the exact three prefixes.
