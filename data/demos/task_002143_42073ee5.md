This is a Robust PCA (RPCA) task — do not substitute a simpler denoising heuristic. First load `.wav` files (e.g. with `scipy.io.wavfile` or `librosa`) and compute the STFT magnitude spectrogram with `scipy.signal.stft` or `librosa.stft`. Implement RPCA via Inexact ALM (or ADMM): iteratively (a) solve for the low-rank matrix L by singular-value thresholding (SVD + soft-threshold singular values), (b) solve for the sparse matrix S by element-wise soft-thresholding of (spectrogram - L), (c) update the Lagrange multiplier, and (d) check convergence via the Frobenius norm of the residual (spectrogram - L - S) against a strict threshold like 1e-5 — stopping too early or using a fixed iteration count instead of a real convergence test is the most common way this goes wrong. Tune the sparsity regularization parameter lambda (typically `1/sqrt(max(m,n))`) rather than hardcoding an arbitrary value. After convergence, compute `sum(L**2)` vs `sum(spectrogram**2)`; if the low-rank energy ratio exceeds 15%, classify "corrupted", else "clean".
