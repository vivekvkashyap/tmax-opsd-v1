First recover the port and token by running `tesseract /app/server_config.png stdout` and parsing the `ListenPort:` and `AdminToken:` lines from the OCR text — don't hardcode guessed values, read what OCR actually returns. Build the server (Flask or `http.server`) bound to the extracted port. For auth, compare the `Authorization` header against `Bearer <token>` and return 401 on mismatch/absence before touching the body. For zip-slip protection, before extracting any member, resolve `os.path.realpath(os.path.join(target_dir, member.filename))` and verify it still starts with `os.path.realpath(target_dir) + os.sep`; reject the whole request (400) if any entry fails this check — checking for a literal `"../"` substring alone is insufficient since absolute paths or `..` combined with symlink-like normalization can bypass naive string checks. Wrap the extraction in `fcntl.flock` on `/app/safe_storage/extraction.lock` (LOCK_EX ... LOCK_UN) to serialize concurrent requests. After extraction, `os.walk` the storage directory, read each file's first 4 bytes, and delete any starting with `\x7fELF`; keep a running counter of retained files across all calls to return in the JSON response. Test each endpoint behavior manually with `curl` before declaring done.
