Do the five steps in order against the literal paths. (1) `mkdir -p .../extracted` and `tar -xzf incoming.tar.gz -C .../extracted`. (2) Detect ELF binaries by content, not extension: iterate files under `extracted/` and test each with `file` (look for "ELF") — a robust loop is `find extracted -type f` piped through a check on `file "$f"`, then `mv` each match into `binaries/` (flat, so if two share a name be careful, but here names differ). (3) Find `.csv` files and convert only the ISO-8859-1 ones: run `file -i` (or attempt `iconv -f ISO-8859-1 -t UTF-8`) and write the converted bytes back to the same path — use a temp file then replace, since `iconv -o` cannot safely read and write the same file in one pass. (4) Read `project_version` from `metadata.json` with `jq -r .project_version` (or python json) and write just that value to `version.txt` with no extra whitespace. (5) Zip so the archive root holds the extracted contents, not the folder: `cd .../extracted && zip -r ../cleaned_project.zip .`. Verify the ELF files are gone from `extracted/` before zipping.
