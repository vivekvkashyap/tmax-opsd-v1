For inactive logs: open the archive in append+read-binary or text mode (`'a+'`), call `fcntl.flock(fh.fileno(), fcntl.LOCK_EX)` before writing, write the header `--- <absolute path> ---\n` followed by the file's exact contents (use `os.path.abspath`), then `fcntl.flock(fh.fileno(), fcntl.LOCK_UN)` after writing and before closing. Read the log's content first (before truncating) so you can append it faithfully, then truncate the original file to 0 bytes — use `open(logpath, 'w').close()` or `os.truncate(logpath, 0)`, not delete/unlink. Watch two easy mistakes: don't add an extra newline if the content already ends in one, and make sure you finish reading/writing the log's own content before truncating it (truncate only after the archive write succeeds, so an active-log detection race doesn't lose data). Run the script once so the archive and truncated files exist on disk for verification.
