Set up the Go module first (`go mod init mle/variance` in `/home/user/src`, then `go get gonum.org/v1/gonum/...`) so imports resolve before writing logic. Read the CSV with `encoding/csv`, parsing each row into a float64 slice, and load it into a `mat.Dense` (rows=samples, cols=features). Center the data by computing the per-column mean (sum column / n) and subtracting it from every entry in that column — do this with an explicit loop or `mat.Dense.Apply`, not by centering per-row by mistake. Compute the sample covariance matrix with N-1 (not N) degrees of freedom: `Cov = (Xc^T * Xc) / (n-1)` where Xc is the centered matrix — gonum's `stat.CovarianceMatrix` can do this directly if you pass it the raw (uncentered) data, or you can do the centering+multiply manually and divide by n-1. Use `mat.Eigen` (or `EigenSym` since covariance matrices are symmetric — this is more numerically robust and gives real eigenvalues directly) to get eigenvalues, extract the real parts, sort descending, and write each formatted to exactly 4 decimal places (`fmt.Sprintf("%.4f", v)`) on its own line to the output file. Create the output directory before writing if it doesn't exist.
