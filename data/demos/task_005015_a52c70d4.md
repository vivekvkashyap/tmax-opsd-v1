Test the hook directly by pushing a dummy commit before trusting the Expect layer. For automation, write `trigger.exp` using `spawn`, then three `expect "prompt text"` / `send "answer\r"` pairs matching the exact prompt strings from `make_alert.sh`, ending with `expect eof`.
