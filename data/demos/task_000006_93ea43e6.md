Write a Python script that walks the archive tree programmatically rather than manually unzipping by hand — use `glob`/`os.walk` to find every `.zip` under the backups directory, open each with `zipfile.ZipFile`, iterate its members in memory, and for each `.tar.gz` member open it via `tarfile.open(fileobj=..., mode='r:gz')` on the extracted bytes (no need to write intermediate files to disk). For each `.wal` member inside, read its text content and parse each line by splitting only the first four whitespace-separated tokens (timestamp, service, operation, key) and treating everything remaining as the value — this matters because values can contain spaces, so a naive `split()` truncates multi-word values. Collect all matching `api_gateway` entries from every archive into one list of (timestamp, op, key, value) tuples, then sort that combined list by integer timestamp before replaying: SET upsizes the key, DELETE removes it if present, and later timestamps always win regardless of which archive/tarball they came from. Only after processing the fully sorted, merged event stream should you dump the final dict to JSON with `indent=4`.
