Use `h5py` to read the `A` and `s` datasets from `/home/user/input.h5`. Compute the degree diagonal with `D = np.diag(A.sum(axis=1))` and the Laplacian `L = D - A`. Get the SVD via `numpy.linalg.svd(L)` — note it already returns singular values in descending order, so the "top 5" are simply the first 5 entries of `S` and the first 5 columns/rows of `U`/`Vt` (no extra sorting needed). Reconstruct the low-rank approximation as `U[:, :5] @ np.diag(S[:5]) @ Vt[:5, :]` — be careful with matrix dimensions and that you're slicing `U`'s columns and `Vt`'s rows, not the reverse. Then integrate `dx/dt = -L_approx x` explicitly: starting from `x = s.copy()`, repeat the update `x = x - dt * (L_approx @ x)` for exactly `int(1.0/0.1) = 10` steps (dt=0.1) — do not use `t=0` to `t=1.0` inclusive of an 11th step; the number of Euler steps is what verification checks numerically to tight tolerance. Save the final `x` to `/home/user/output.h5` under the dataset name `x_final` using `h5py.File(..., 'w')` and `create_dataset`. Run the script once at the end so the output file actually exists on disk.
