For each kept file, branch on its extension: for `.bin` run `readelf -h <file> | grep 'Entry point address' | awk '{print $NF}'` to get the address string verbatim; for `.gcode`, grep the `; estimated printing time =` comment line, parse the `Xh Ym Zs` pattern with a regex, and compute `X*60 + Y + Z/60` truncated to an integer; for `.wal`, read the first 4 bytes with `xxd -p -l 4 <file>` (or `od`) to get the lowercase hex magic number. Build the CSV rows and sort by FilePath (`sort -t, -k1,1`), and separately generate `sha256sum` over the same file list sorted the same way — don't modify the deployed files' permissions or contents while inspecting them.
