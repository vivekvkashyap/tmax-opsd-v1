In Go, unmarshal devices.json into structs and build a map of active device IDs; parse the headerless sensor CSV with encoding/csv, discarding rows for devices that aren't active or aren't present in the device list. Group readings by device, sort each group by timestamp, then fill blank temperature entries via linear interpolation between the nearest valid readings before and after the gap (track the last known point and scan forward to the next known one when you hit an empty string). Compute a 3-point trailing simple moving average per device on the interpolated series, special-casing the first two indices to average only the points available so far. Pool all devices' points, group by region, and within each region select the top 2 by smoothed_temp descending, applying the tie-break (older timestamp, then device_id alphabetically) only on exact ties. Marshal the final list — sorted by region ascending, then smoothed_temp descending within region — to top_sensors.json, then build and run the program so the file is actually produced.
