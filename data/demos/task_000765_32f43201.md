Work in a single awk/sort pipeline rather than trying to do it all in one command. First strip the header and sort numerically/lexically by the Date column (`tail -n +2 file | sort -t, -k1,1`) since ISO dates sort correctly as strings. Then do a single forward pass with awk (or a small script) that tracks the last-seen non-empty Sales value and substitutes it whenever the field is empty — this is the classic forward-fill pattern: `last = ($2=="" ? last : $2); print $1, last`. Keep a small sliding window (array or three variables) of the last up-to-3 imputed values as you iterate to compute the rolling average; use integer division (floor) explicitly rather than trusting awk's default float formatting — cast with `int()` after summing and dividing. For the first two rows, average only over the values seen so far (1 then 2 values), not a fixed window of 3. After computing rolling averages for every row, sort by rolling average descending, breaking ties with ascending date (`sort -t, -k2,2nr -k1,1`), then take the top 3 with `head -n 3` and write exactly `Date,RollingAvg` with no header to the output path.
