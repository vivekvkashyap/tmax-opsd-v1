Build the pipeline with sklearn's `ColumnTransformer` combining two `Pipeline` branches: numeric columns through `SimpleImputer(strategy='mean')` -> `StandardScaler()`, categorical columns through `SimpleImputer(strategy='constant', fill_value='missing')` -> `OneHotEncoder(handle_unknown='ignore')`. Wrap this preprocessor and a `RandomForestClassifier(random_state=42)` (name this final step e.g. `classifier`) in an outer `Pipeline`. Feed that whole pipeline into `GridSearchCV(pipeline, param_grid, cv=3)` where grid keys must be prefixed with the step name, e.g. `classifier__n_estimators` and `classifier__max_depth` — a common mistake is forgetting the `stepname__` prefix, which silently causes sklearn to raise or ignore the params. Fit on the full X (all columns except `is_promoted`) and y. Save the fitted `grid_search.best_estimator_` (the whole pipeline) via `joblib.dump` to the pkl path, and write `best_cv_score`/`best_params` from `grid_search.best_score_`/`best_params_` to the metrics JSON, keeping `None`/`null` for max_depth as JSON null. Run the script once so both output files actually exist on disk.
