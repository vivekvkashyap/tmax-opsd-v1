Structure the script around three careful stages rather than one pass. First parse `/home/user/config/backup_system.ini` with `configparser`, reading the `[manifests]` section as pairs of manifest path -> encoding name. Second, for each manifest, open it with the *specified* encoding (not always UTF-8 — a naive attempt hardcodes UTF-8 and mangles non-ASCII bytes) and split each line into a relative path and a size; decode the path properly so any non-ASCII characters come through as correct UTF-8 in the final JSON. Third, for Zip Slip detection, resolve each candidate path as if joined under `/home/user/extract/` using something like `os.path.realpath(os.path.join(base, rel_path))` and check with `os.path.commonpath` (or a prefix check on the resolved absolute path plus a trailing separator) that the result still starts with the resolved `/home/user/extract/` — string-prefix checks without proper path resolution are a classic bug here (e.g. `/home/user/extract_evil` would falsely pass a naive prefix test). Accumulate total bytes and safe path names only for non-malicious entries, count the rest as skipped, sort `safe_files` alphabetically, then write JSON to a temp file in the same directory and atomically `os.replace()` it onto `/home/user/safe_extraction_plan.json`.
