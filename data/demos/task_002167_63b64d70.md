For free-space checking, use `shutil.disk_usage('/home/user/data').free` (or `os.statvfs`) rather than parsing `df` output — it's more portable and avoids shell parsing bugs. Wrap the spawn+wait in a retry loop capped at 5 attempts total; only increment log/retry state on nonzero exit, and stop as soon as exit code is 0. Capture the last non-empty line of `child.before` (excluding the prompt echo) for the success log line.
