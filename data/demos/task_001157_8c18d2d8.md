The core difficulty flagged in the task is that transcript text can contain commas inside quoted fields — a plain `awk -F','` or `cut -d','` will split those transcripts into extra fields and shift start_ms/end_ms. Instead, parse it properly in awk using `FPAT` (e.g. `awk -v FPAT='[^,]*|"[^"]*"'`) or use a small state machine that only splits on unquoted commas, so quoted commas stay inside the transcript field. In build_manifest.sh, strip quotes from the transcript, trim whitespace, and compute duration_ms as end_ms - start_ms with integer arithmetic, numbering segment_id from 1.
