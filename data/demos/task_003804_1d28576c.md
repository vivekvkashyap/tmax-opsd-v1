Once you have both values, implement the pipeline as: read raw bytes from `sys.stdin.buffer.read()`, XOR every byte with the recovered key, then standard-Base64-encode the XORed bytes (`base64.b64encode`), and finally remap each character of the standard alphabet (`A-Z a-z 0-9 + /`) to the corresponding position in the custom alphabet you recovered — the cleanest way is `str.maketrans` between the standard 64-char alphabet string and your recovered custom one, then `.translate()` the base64 output. Print the result with `sys.stdout.write` (no trailing newline).
